Implement a working Breed dropdown system for the Chat Room selection area in ScrollPet.
________________________________________
Breed Dropdown – General Rules
•	The breed dropdown appears in the LEFT COLUMN top section (pet & breed selector).
•	Dropdown label:
o	Default: All Breeds
o	After selection: selected breed name
•	Breed options must change based on selected pet type.
•	Selecting a breed filters the chat room to:
•	Pet Type → Selected Breed → Location
•	Selecting All Breeds resets breed filtering.
•	Breed selection must NOT affect Global or Country logic, only chat grouping.
